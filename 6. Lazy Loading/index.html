📚 Giới thiệu về Lazy Loading

Lazy Loading là kỹ thuật trì hoãn việc tải hình ảnh cho đến khi chúng thực sự cần thiết (khi user scroll đến vùng hiển thị). Điều này giúp:

  • ⚡ Giảm thời gian tải trang ban đầu
  • 💾 Tiết kiệm băng thông
  • 📱 Cải thiện trải nghiệm trên mobile
  • 🚀 Tăng Core Web Vitals scores

Native Lazy Loading

Sử dụng thuộc tính loading="lazy" của HTML5

✅ Ưu điểm:

  • Đơn giản, không cần JavaScript
  • Browser tự quyết định

❌ Nhược điểm:

  • Khó control loading behavior
  • Không có Custom animations/Effects
🚀 Xem Demo Native
🔍

Intersection Observer

Sử dụng JavaScript Intersection Observer API

✅ Ưu điểm:

  • Full control loading behavior
  • Xử lý Custom animations/Effects

❌ Nhược điểm:

  • Phức tạp hơn
  • Cần JavaScript, Tự implement logic

📚 Giới thiệu Native Lazy Loading

Native Lazy Loading là tính năng built-in của browser, chỉ cần thêm loading="lazy" vào thẻ <img>.

Browser sẽ tự động trì hoãn việc tải hình ảnh cho đến khi chúng gần vào viewport.

🔍 Hướng dẫn Test

  1. Mở DevTools (F12) → Tab Network
  2. Reload trang để clear cache
  3. Quan sát: Chỉ có vài ảnh đầu được load
  4. Scroll xuống để thấy ảnh load từng batch
  5. Toggle Lazy Loading để so sánh
